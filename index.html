    Awesome Linux GUI

    For many years, Linux GUI users struggled to find software that was both modern, easy to use, and powerful—unlike users of other operating systems. However, things have changed: in recent years, many great apps designed for or available on Linux have emerged. This repo aims to provide a curated list of awesome Linux GUI applications, inspired by the awesome-mac project.
